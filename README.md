# Учебный проект "Научиться учиться"  

## Описание
Первый учебный проект в Яндекс Практикуме. Одностраничный сайт-лендинг, рассказывающий о различных эффективных способах обучения.  

## Функционал
Разметка и верстка страницы  
Методология БЭМ 
Флексбокс-верстка 
Фреймы 
Кейфреймы  
Анимация объектов
